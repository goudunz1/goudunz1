#### goudunz1's GitHub

Welcome to my GitHub page. This is where I store most of my projects, writeups
and exploit scripts. I'm currently playing CTF games for Team Trinity at
Nanjing University. Contact me at [goudunz1@outlook.com](mailto:goudunz1@outlook.com).

These're some of the projects that I contributed to / am currently working on:

- [varviewer](https://github.com/goudunz1/varviewer) a tool aimed at precisely
  matching variables between source code and binaries with the help of DWARF
  debugging information

  Varviewer can act as an auxiliary tool for Ghidra and IDA Pro to improve the
  results of disassembling and decompilation.

- [Maoware](https://github.com/goudunz1/Maoware) a simple multi-platform malware
  that implements modern techniques, inspired by a cyber attack at Nanjing
  University in June, 2024

- [Snake86](https://github.com/goudunz1/Snake86) a tiny hungry snake game
  written in pure MASM 8086 assembly language that runs on DOS systems
