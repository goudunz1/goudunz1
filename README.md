#### goudunz1's GitHub

Welcome to my GitHub page. This is where I store most of my projects, writeups
and exploit scripts. I'm currently playing CTF games for Team Trinity of
Nanjing University. Contact me at [goudunz1@outlook.com](mailto:goudunz1@outlook.com).

These're some of the projects that I contribute/am currently working on:
- [Snake86](https://github.com/goudunz1/Snake86) Snake86 is a tiny hungry snake
  game written in pure 8086 assembly language that runs on DOS systems.
- [VarViewer](https://github.com/goudunz1/varviewer) VarViewer is a tool aimed
  at precisely matching variables between source code and binaries with the
  help of DWARF debug information. VarViewer can act as an auxiliary tool for
  Ghidra and IDA Pro to improve the results of disassembling and decompilation.
